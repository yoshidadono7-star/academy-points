# HERO'S延岡校 プロジェクト

宮崎県延岡市の学習塾「個別指導学院HERO'S延岡校」の運営に関するWebサイト・資料・システム設計を管理する。オーナーは吉田寛（62歳、早稲田大学商学部卒、産業カウンセラー・NLP・アドラー派コーチング資格保有）。非エンジニア。説明は平易な日本語で、技術用語には補足を添える。

---

## 教育モデル（3本柱）

1. **個別指導（AI個別最適化）** — atama+・edu+・学びエイドなどAI教材による個別最適化学習
2. **フォーカスタイム** — ポモドーロ形式（25分学習＋5分休憩）の自立学習。SRJ速読解力講座を冒頭20分に実施
3. **コーチング（1on1）** — アドラー派コーチング＋NLPベースの週次面談。GROWモデル使用

### 使用AI教材・ツール
| ツール | 用途 |
|--------|------|
| atama+ | 数英理社国のAI個別最適化（オプション） |
| edu+ | 映像授業（オプション） |
| 学びエイド | 映像授業（オプション） |
| Monoxer | AI記憶定着（オプション） |
| SRJ速読解力講座 | 速読トレーニング（4講座：速読解力・速読聴英語・新国語・算数的思考力） |
| Manabo | 常時質問対応 |
| Comiru | 保護者連絡・請求・出退席管理の一元化 |

**注意：** ClipTeacher・名大SKYプログラムは導入していない。使用しないこと。

## 作業上の注意事項

- ファイルを編集する前に必ず確認を取ること
- 大きな構造変更は提案→承認→実行の3ステップで進める
- 保護者の目に触れる資料は、誤字脱字ゼロを目指す
- gamification用語（ブースター、チームチャレンジ、レイドボス等）は未実装なのでHP・外部資料には使わない
- 「スマホ封印ルール」は表現しない。代わりに「誘惑のない集中環境」「集中できる環境設計」と書く
- SRJ4講座（速読解力・速読聴英語・新国語・算数的思考力）は速読解力と別の3講座を分けて記載する
- ファイル成果物はWord（.docx）とPDF（.pdf）の両方を作成・納品する
- HP用写真ファイル（hero-*.jpeg）はHTMLと同一ディレクトリに配置する
