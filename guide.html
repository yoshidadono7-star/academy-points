管理画面

管理者専用
ログイン方法: メールアドレス (admin@heros.jp) + パスワード
URL: hero-s-points.web.app

  • デイリー

    今日のプラン確認、バディチャット、学習モニター（リアルタイム状況）、生徒コンディション確認、メッセージ送受信を行います。

  • 生徒管理

    生徒の登録・編集、教材進捗の確認、学習ルート設定、学習履歴の閲覧、テスト結果の記録、面談記録の管理ができます。

  • ゲーム

    ランキング表示、ギルド管理、レイドボス設定、ストーリー管理、ミッション・イベント作成、タスク管理を行います。

  • 分析

    生徒のリスクモニタリング（学習ペースの低下検知）やデータ分析ダッシュボードで、教室全体の状況を把握します。

  • 運営

    システム設定、ポイント付与テンプレート、集団目標の管理など、教室運営に関する設定を行います。

ヒント: 管理画面は多機能のため、まずは「デイリー」と「生徒管理」を中心に使い始めるのがおすすめです。

生徒画面

生徒
ログイン方法: ニックネーム + PIN (4桁)
URL: hero-s-points.web.app/student

ログイン手順

  1. 「ニックネーム」を入力（管理画面で登録した名前）
  2. 4桁のPINコードを入力
  3. 「ログイン」をタップ

  • バディチャット

    AIバディ（ルカ、ミミ、レックスなど8体から選択）と会話できます。勉強の相談や励ましをもらえます。

  • スペースベース

    ポイントを使って宇宙基地に施設を建設できます。学習を頑張るとポイントが貯まります。

  • ポモドーロ学習

    タイマーを使った集中学習モード。完了するとポイント・EXP・ガチャチケットがもらえます。

  • アバター・バディ選択

    自分のアバターをカスタマイズしたり、お気に入りのバディを選んだりできます。

ヒント: ポモドーロ（25分の集中学習）を完了すると、RPGゲームで使えるチケットとゴールドがもらえます。

講師画面

講師
ログイン方法: 講師PIN (4桁)
URL: hero-s-points.web.app/teacher

  • 今日のスケジュール

    本日の授業予定と担当生徒を確認できます。

  • 生徒コメント記録

    授業後に生徒ごとのコメントを記録します。科目・態度・理解度を選択し、バディ共有のオン/オフも設定できます。

  • AI相談

    AIバディに指導方法のアドバイスを相談できます。��徒の状況に応じた対応のヒントが得られます。

  • 家庭学習承認

    生徒が提出した家庭学習の成果を確認・承認します。

  • 引継ぎ受信箱

    他の講師からの引継ぎ事項を確認できます。

  • 目標承認・シフト確認・統計

    生徒の目標設定の承認、自分のシフト確認、指導統計の閲覧ができます。

オーナー画面

オーナー
ログイン方法: メールアドレス + パスワード (Firebase Auth)
URL: hero-s-points.web.app/owner

  • KPIダッシュボード

    教室数、総生徒数、総講師数、未読の引継ぎ件数をカードで一覧表示します。

  • 教室切替

    複数教室を管理している場合、教室カードをタップして切り替えられます。

  • オーナーバディAI

    経営に関する相談をAIバディに行えます。教室・生徒・講師・引継ぎログをもとにアドバイスを提供します。

  • 引継ぎ受信箱

    各教室の講師からの引継ぎ・アラートを確認できます。

RPGゲーム

生徒
ログイン方法: ニックネーム + PIN (4桁)（生徒画面と同じ）
URL: hero-s-points-rpg.web.app

遊び方

  1. 塾アプリでポモドーロ学習を完了してチケットとゴールドを獲得
  2. RPGにログインしてホームタウン（宇宙ステーション）を探索
  3. ショップでガチャを引いて武器・装甲・バディを入手
  4. ミニゲームで腕試し

  • ガチャ（ショップ）

    3種類のガチャがあります。
    スタンダード: チケット1枚で回せます（コモン〜レジェンド）
    プレミアム: プレミアムチケット1枚（アンコモン〜レジェンド、高レア率）
    無料デイリー: 1日1回無料（コモン〜レア）

  • ミニゲーム

    3種類のミニゲームで遊べます。
    MathFlash: 30秒間の計算チャレンジ
    MemoryMatch: 60秒間の神経衰弱
    TypingHero: 45秒間のタイピングチャレンジ

  • ホームタウン

    宇宙ステーションの施設をタップして探索できます。レベル・EXP・ゴールドなどのステータスもここで確認できます。

大事なルール: RPGは1回のプレイが最大2分。勉強の邪魔をしない「ごほうびゲーム」です。勉強を頑張るほど強くなれます。
